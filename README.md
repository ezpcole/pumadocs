# pumadocs
this repository serves as a base for most documentation/information that robotics teams under the Pumabots organization may need  
this info is also available as a resource for any other teams if they wish

## how to build

1. clone the github repository
2. then open the python virtual environment by running `.venv/Scripts/activate`
3. then run `sphinx-build -M html docs/source docs/build`

## how to contribute

feel free to open issues to anything that needs to be addressed, and make pull requests if you wanna change anything  
if you are in the org, please tell Eza in person if you wish to have something addressed quicker, as this repo isn't checked constantly nor are his emails
